v1.1.0 brings social login options and powerful admin tools.

## OAuth Providers

Sign in with GitHub or Google. Configure by adding your OAuth credentials to `.env`.

## Admin Features

Super admins can now impersonate users for support purposes. All impersonation sessions are logged with timestamps and reasons.
