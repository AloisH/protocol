# Bistro Documentation

Welcome to the Bistro documentation. Learn how to build AI-powered SaaS applications with our free, open-source starter kit.

## Quick Start

Get started with Bistro in minutes:

1. Clone the repository
2. Install dependencies with `bun install`
3. Set up your environment variables
4. Start the development server with `bun dev`

## Core Features

- **Authentication** - Email/password + OAuth providers (GitHub, Google)
- **Multi-tenancy** - Organizations with member management
- **Database** - Prisma 7 with PostgreSQL
- **Content** - Blog and documentation with Nuxt Content
- **AI Integration** - Ready for Vercel AI SDK (coming soon)
