# Welcome to Bistro

Bistro is a free, MIT-licensed Nuxt 4 starter kit for building AI-powered SaaS applications.

## Why Bistro?

Expensive SaaS starters cost $149-$349. We believe great tools should be accessible to everyone.

## Features

- **Modern Stack:** Nuxt 4, Nuxt UI v4, Tailwind 4
- **Authentication:** Better Auth with email/password + OAuth
- **Database:** Prisma 7 + PostgreSQL
- **Multi-tenancy:** Organizations with role-based access control
- **Content:** Nuxt Content for blog and documentation
- **Testing:** Vitest + Testing Library
- **CI/CD:** GitHub Actions ready
- **Docker:** Production-ready containers

## Getting Started

Check out our [documentation](/docs) to get started with Bistro.

```bash
# Clone the repository
git clone https://github.com/AloisH/bistro.git

# Install dependencies
bun install

# Start development server
bun dev
```
