Bistro v1.0.0 marks our first stable release. This version includes everything you need to build a production-ready SaaS application.

## Highlights

- **Authentication**: Secure email/password login with session management
- **Multi-tenancy**: Create organizations and invite team members
- **Todo Example**: Full CRUD implementation as a feature template
- **Modern Stack**: Nuxt 4, Vue 3, Tailwind 4, PostgreSQL
